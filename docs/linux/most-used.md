# Most useful Linux comands
A (non-comprehensive) list of linux commands to help with general use.

# Must know
`ls` - List files

| **Command** | **Explanation** |
|:-----------:|:---------------:|
|`ls`             |List files in directory                 |
|  `cd`          |Change directory                 |
|`pwd`|Prints name of current directory|
